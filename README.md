test-streaming
==============

Tiny experiences and prototypes with WebRTC & Node.Js streams.
Currently, it's a kind of "audio conference" web room.

Installation
------------

Needs : NPM & Node.js > 0.10

```
npm install
node app
```

Visit `http://localhost:3030`
